# FARISCO-Ltd.github.io

صفحة روابط احترافية لـ FARIS | فارس.

## التعديل السريع

عدّل الروابط فقط من ملف `config.js`:

- Creators
- PayPal
- TikTok
- Telegram

## النشر على GitHub Pages

1. أنشئ Repository باسم `FARISCO-Ltd.github.io`.
2. اجعله Public.
3. ارفع كل الملفات الموجودة داخل هذا المجلد.
4. اضغط Commit changes.
5. افتح الرابط: `https://farisco-ltd.github.io`.



## ملاحظة مهمة بعد تحديث الشعار

تم تعديل `sw.js` حتى لا يحتفظ بالشعار القديم في الكاش.
بعد رفع النسخة، افتح الموقع مرة واحدة بهذا الرابط:

`https://farisco-ltd.github.io/?v=4`

ثم بعد ذلك استخدم الرابط العادي:

`https://farisco-ltd.github.io/`
